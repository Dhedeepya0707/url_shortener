KeepItShort | Modern URL Shortener
KeepItShort is a high-performance, full-stack web application designed to transform long, cluttered URLs into clean, manageable short links. Built with a robust Java Spring Boot backend and a sleek, responsive frontend, it utilizes MySQL for persistent data storage.

 Features
Instant Shortening: Convert any valid URL into a unique short code in one click.

Persistent Storage: All links are securely stored in a MySQL database.

Modern UI: A dark-themed, glassmorphism-inspired interface for a premium user experience.

CORS Enabled: Seamless communication between the frontend and backend services.

Tech Stack
Backend: Java 21, Spring Boot 4.1.0 (Snapshot), Spring Data JPA.

Database: MySQL 9.6.

Frontend: HTML5, CSS3 (Modern Flexbox), JavaScript (Async/Fetch API).

Build Tool: Maven.
